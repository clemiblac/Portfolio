COVID in the United States

This project explores COVID in the United States and compared data from The New York Times and The Atlantic to understand data reliability and integrity of reported COVID data. On this site, I created the following charts using JavaScript: [State] Reported Cases: Top Ten Counties, [State] Reported Deaths: Top Ten Counties, Re-arrangeable table of “US Counties with Highest Total Cases”, Both Honeycomb/ Tile Diagrams, Re-arrangeable New York Times and Atlantic Data Comparison Table, Data Reliability Donut Chart, Both Time Series Charts with buttons

Crime in Washington, D.C.

The purpose of this project was to explore Washington, D.C. crime data from the open D.C. database, visualize the data and apply machine learning to the data. For this project, I was responsible for building the website using HTML,bootstrap and CSS. I also built all 45 graphs in the Daily Rhythm of crime chart and sunburst chart using JavaScript. Additionally,I developed the flask app that pulled the information from database for the web visualizations using Python and mySQL.

SQL Alchemy API

The purpose of this project was to use SQL Alchemy to query a database containing precipitation, temperature and station data. The date range is from 2010-01-01 to 2017-08-23. Using the available routes, you can copy and paste the urls to view the data
